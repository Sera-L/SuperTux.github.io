# SuperTux Redesigned Website [DEMO] 

The SuperTux website is generated automatically from the content in this
repository using [Jekyll](https://jekyllrb.com/) and then published using
[GitHub Pages](https://pages.github.com/).

To test it locally in your computer or server make sure you've installed **ruby** and **jekyll** then run the following command in the terminal
`jekyll serve`
and type in your browser
`http://localhost:4000`
